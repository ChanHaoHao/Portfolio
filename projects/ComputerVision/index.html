Computer Vision

Project Overview

This coursework covers core concepts and practical implementations in computer vision. Each task involved hands-on development of classical and deep learning techniques for image analysis, recognition, and generation. The projects demonstrate proficiency in image filtering, frequency analysis, homography estimation, panoramic stitching, neural network training, and modern generative models such as diffusion networks.

Skills Used

PID Image filtering Metric learning model Diffusion models OpenCV Scipy SIFT keypoint extraction Panorama Gaussian/Laplacian pyramids PyTorch

Image Filtering

This assignment focused on foundational image processing techniques using spatial filters. I implemented gradient, Gaussian, box, and steerable filters from scratch and visualized their effects on real images. Additionally, I explored the impact of filter ordering and conducted quantitative analysis of edge detection errors. Median filtering was also applied to denoise images, with performance measured using mean squared error.

Frequency Domain Analysis

In this homework, I investigated the use of the 2D Discrete Fourier Transform (DFT) for image analysis. I applied the Convolution Theorem to compare spatial and frequency-domain filtering, constructed Gaussian and Laplacian pyramids, and implemented a multi-level image reconstruction pipeline. The assignment concluded with Laplacian pyramid blending, where I seamlessly combined images using spatial-frequency representations.

I have blended the mango and pineapple together, making the boundary nearly invisible.

Recognition and Metric Learning

This project involved building and training deep learning models for face recognition. I implemented classification using a pretrained ResNet18, and contrastive metric learning using a memory bank and cosine similarity. I also evaluated models with t-SNE visualization, and reflected on the generalization and robustness of different learning paradigms.

Panorama Stitching

In this assignment, I developed a complete panoramic image stitching pipeline. The process included keypoint detection using SIFT, descriptor matching with Lowe’s ratio test, and robust homography estimation using RANSAC. I applied perspective warping and image blending to generate seamless panoramic images from multiple overlapping views of real-world scenes.

I stitched the three images above by detecting the features in each image with SIFT, and find the homography by RANSAC.

Diffusion Models

This final project explored diffusion models for generative tasks. I implemented forward and reverse sampling processes, visualized noise progression, and built iterative denoising loops using pretrained DeepFloyd models. Tasks included denoising, image inpainting, and guided generation using prompt embeddings and Classifier-Free Guidance (CFG), offering insights into state-of-the-art generative AI. Also
